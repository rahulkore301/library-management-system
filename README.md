# 📚 Library Management System

A full-stack Library Management System built using Node.js, Express, MySQL, and deployed on AWS EC2 with Nginx.

---

## 🚀 Live Demo
🌐 http://52.206.220.24

---

## 🧰 Tech Stack

### Frontend
- HTML5
- CSS3 (Modern UI)
- JavaScript (Fetch API)

### Backend
- Node.js
- Express.js

### Database
- MySQL

### Deployment
- AWS EC2 (Ubuntu)
- Nginx (Web Server)

---

## ✨ Features

- 🔐 Admin Login System
- 📊 Dashboard with Analytics
- 📚 Add Books
- 📖 View Books
- 👥 Manage Users
- 📕 Issue Books
- 🔄 Real-time Data Fetch from API
- 🌐 Deployed on Cloud (AWS)
